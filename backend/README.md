# Backend

FastAPI backend for the food-delivery marketplace.

## Owner

Team Lead / Backend Core is responsible for:

- FastAPI application structure
- REST API endpoints
- Swagger/OpenAPI documentation
- PostgreSQL integration
- Redis integration
- WebSocket live order status
- Token Bucket Rate Limiter integration
- Settlement batch logic support
